# Flash Cards Pro 🎯

A modern, mobile-first Progressive Web App for studying with flashcards.

## Features
- 📱 Install as mobile app
- 🌐 Works offline
- 🎨 Dark/Light theme
- 🧠 Spaced repetition algorithm
- 🎯 Multiple study modes
- 🔊 Text-to-speech support
